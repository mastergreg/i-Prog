
Η εφαρμογή έχει ένα καμβά όπου ζωγραφίζεται ένας κύκλος. Σε κάθε mouse click
event ο κύκλος ζωγραφίζεται και πάλι με κέντρο το σημείο όπου ο συνέβει το event.
Επιπλέον, η εφαρμογή κάθε N milliseconds τοποθετεί τον κύκλο στο κέντρο του καμβά.
Υπάρχει μάλιστα ένα text-field όπου ο χρήστης μπορεί να δίνει το N.  


CircleApp ::
	το class που είναι το παράθυρο της εφαρμογής (ξεκινήστε εδώ).
	
Canvas ::
	το class που υλοποιεί τον καμβά με τα απαιτούμενα events και event
	handlers.

CircleReset ::
	το class που υλοποιεί το thread που τοποθετεί τον κύκλο στο κέντρο
	του καμβά κάθε N milliseconds.
